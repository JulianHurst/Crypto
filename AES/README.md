**aes** permet le chiffrement d'un bloc de 128 bits par AES avec les opérations :
* SubBytes()
* ShiftRows()
* MixColumns()
* AddRoundKey(int r)

Et le déchiffrement avec les opérations inverses :
*  Inv_SubBytes()
* Inv_ShiftRows()
* Inv_MixColumns()

Il n'y a pas d'arguments d'entrée ni de fichier de sortie.
